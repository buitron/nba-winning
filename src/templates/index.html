This web application is a lightweight scrapping app that retreives current NBA team stats and runs it though a pre-trained supervised machine learning algorithm, the results are predicted end of season team win values. The model is most accurate when executed at the end of the regular season. Click on the button to the right to start the app ** might take sometime to scrape the data **

As an added bonus I am including a live streaming scoreboard. Click on the toggle-it button to view live game data

No Current Games: This application is still under development.
